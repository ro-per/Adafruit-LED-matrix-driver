## 1. Effects                             <a name="I"></a>
### 1.A Possibles Modes
- [x] Show a picture ```arg[1]=<picture>.ppm``` (lab 3)
- [x] Show a text ```arg[1]=<text>.txt``` (killer feature)
### 1.B Possibles Effects 
(pass to arg[x] after passing mode arguments)
#### Color Effects
- [x] ``` --colors=grey ```
- [x] ``` --colors=invert ```
- [x] ``` --colors=gamma ```
#### Mirror Effects
- [x] ``` --mirror=vertical```
- [x] ``` --mirror=horizontal ```
#### Other Effects
- [x] ``` --scroll ```
